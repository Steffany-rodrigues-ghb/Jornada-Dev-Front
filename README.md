# Jornada-Dev-Front
Este repositório documenta meu desenvolvimento como front-end, reunindo práticas, mini projetos e anotações sobre HTML, CSS e JavaScript conforme avanço no aprendizado das tecnologias base da web.

# Objetivo:
Registrar minha evolução como desenvolvedor front-end por meio de:
- Exercícios práticos;
- Projetos de estudo;
- Testes, anotações e experimentos;
- Implementação gradual de conceitos do nível básico ao avançado;

# Tecnologias Estudadas :
- HTML5 – Estrutura e semântica;
- CSS3 – Estilização, layouts e responsividade;
- JavaScript – Lógica, interações e manipulação de elementos;

# Sobre este Repositório
Este repositório tem como finalidade documentar minha trajetória de aprendizado em desenvolvimento front-end. Ele servirá como referência para acompanhar meu progresso técnico, organizar meus estudos e registrar o conhecimento adquirido ao longo do processo.
